# DoctorBot

This project is developed for MIT ADT University, as a part of mini project.

## Getting Started


### Prerequisites

NLTK - Natural Language Toolkit

```
pip install nltk
```

sklearn

```
pip install sklearn
```

gTTS - Text to speech

```
pip install gTTS
```

winsound - Playing sound in Windows

```
pip install winsound
```

PyQt5 - User Interface
```
pip install PyQt5
```
